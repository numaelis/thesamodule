Thesa Module

v 1.2

Tryton Module to serve qml files (tabs) for thesa.

how to use:

After installing the module in trytond, copy the folders with tabs files(qmls) en qml folder and from tryton click button update all qml folders.

Then from thesa upon login choose "type system" = module.

different types of files can be served not only qml.

subdirectories are not currently supported in the folders.

Starting from version 1.2, multiple folders are allowed for users selection.
also the default folder
the different folders are placed in the qml folder looking like this
  -qml
      - default
      - foldernamexxx
      - ....
